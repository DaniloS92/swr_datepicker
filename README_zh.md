# swr_datepicker

[README](README.md) | [中文文档](README_zh.md)

## 什么是swr_datepicker?

Odoo原生日期控件的扩展，支持月份和年份的显示

## 示例

举例: month 和 year 是日期型字段

显示成月份:

&lt;field name="month" options='{"minViewMode": "months", "viewMode": "months", "format": "YYYY年MM月"}'/&gt;
